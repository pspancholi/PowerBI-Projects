

# 📊 Mental Health & Social Media Analytics Dashboard

## 🚀 Overview  
This project is my **first Power BI dashboard**, where I analyzed the impact of social media usage on students' **mental health, academic performance, and lifestyle habits**.  
It focuses on transforming raw data into meaningful insights using data modeling, DAX, and interactive visualizations.

---

## 📁 Dataset & Modeling  
The project uses two datasets: **Student Details** and **Platform Details**, connected using `Student_ID`.  
A **star schema** was implemented, with student demographics as dimensions and usage behavior as fact data.  
A custom date table was also created for time-based analysis.

---

## 📊 Key Insights  
- Students spend approximately **5 hours per day** on social media  
- Around **64% of students** reported an impact on academic performance  
- Increased usage is associated with **reduced sleep duration**  
- **Undergraduate students** show relatively higher addiction levels  
- Platform usage patterns vary across age and academic levels  

---

## ⚙️ Dashboard Features  
- Executive overview with key KPIs  
- Mental health and lifestyle analysis  
- Academic impact by platform and usage  
- Relationship and conflict trends  
- Drill-through for individual student profiles  
- Interactive slicers and bookmarks  

---

## 🛠️ Tools & Skills  
- Power BI  
- DAX (Calculated Columns & Measures)  
- Data Modeling (Star Schema)  
- Data Visualization & Storytelling
